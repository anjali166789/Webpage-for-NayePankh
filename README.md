# Webpage-for-NayePankh

A simple and responsive NGO website created for the NayePankh Foundation. This project aims to showcase the organization's mission, initiatives, and impact through a clean and user-friendly interface.

## Features

* Responsive design for desktop and mobile devices
* Hero section with call-to-action
* About Us section
* Our Work section highlighting key initiatives
* Impact statistics section
* Dark Mode toggle using JavaScript
* Smooth and modern UI with CSS animations

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```
NayePankh-Foundation/
│
├── index.html
├── style.css
└── script.js
```

## How to Run

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. Click the Dark Mode button to switch themes.
